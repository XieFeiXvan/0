Detailed Description

Gif 动图

使用方法
  1. load 加载图像
  1. bind 绑定输出 HDC
  1. （可选）setPos 设置位置
  1. （可选）setSize 设置缩放大小（为 0 表示原图大小）
  1. play 开始播放
  1. draw 绘制到 HDC
Bug:
  1. 释放时可能崩溃
  1. 对绑定的 HDC 调整大小可能导致崩溃
